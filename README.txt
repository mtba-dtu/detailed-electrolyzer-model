# detailed-electrolyzer-model
# Online Appendix: Optimization of Hybrid Power Plants: When Is a Detailed Electrolyzer Model Necessary?

Electronic companion for the research paper "Optimization of Hybrid Power Plants: When Is a Detailed Electrolyzer Model Necessary?"
This repository contains the supplementary material for the paper "Optimization of Hybrid Power Plants: When Is a Detailed Electrolyzer Model Necessary?". 
This is joint work by Manuel Tobias Baumhof, Enrica Raheli, Andrea Gloppen Johnsen, and Jalal Kazempour.

In this repository, we provide:
   1. The source code implemented in Julia using Gurobi.
   2. The xlsx file that containts the data of the case study.
